# StringStorage

게임 내에 쓰이는 문자열 데이터의 번역 정보를 저장하는 LevelDb 기반의 데이터베이스입니다.

## Getting Started

### Prerequisites

- .net SDK 9.0

### Installation

download from [github.com](https://github.com/orgs/StudioBside/packages)

```
dotnet add package StringStorage
```
